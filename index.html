TMR Utilities for BYU's SpyDrNet

SpyDrNet-TMR is a package of utilities that applies triplication to netlist designs. After parsing in a netlist with SpyDrNet, the netlist can be triplicated with parameters including which instances/top-level ports will be replicated, options for reduction voters and basic voter insertion algorithms, etc.

This package depends on SpyDrNet, which is a framework for analyzing and transforming netlists.
