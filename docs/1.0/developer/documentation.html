Documentation¶

The SpyDrNet TMR documentation is designed to enhance the user’s experience and empower them to accomplish their goals with all or any of the features that are available to in the suite.

Figures¶

spydrnet_tmr_logo¶

This logo was designed by Dallin Skouson. The source file is spydrnet_tmr_logo.pptx. The PDF is generated from PowerPoint. To vectorize the text in the logo, follow these instructions in Acrobat Reader (from Used to Tech):

  1. Add a blank watermark (some whitespace with 0% opacity).

  2. Use Flattener Preview to flatten the text, make sure to check “Convert All Text to Outlines” and “Convert All Strokes to Outlines” as well.
