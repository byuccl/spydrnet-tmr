Connect Unconn Nets To Dummy¶

spydrnet_tmr.symbiflow.fix_unconn_nets.connect_unconn_to_dummy(netlist)¶

Any output pins that are left unconnected are denoted in the output netlist as being connected to “unconn”. Symbiflow expects all unconnected output pins to be connected to “dummy” wires. This function cycles through the netlist and connects any unconnected output pins to a dummy wire.

Parameters

netlist – the current netlist
