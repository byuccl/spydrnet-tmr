Todos
-----

Sphinx has a `todo extenstion <https://www.sphinx-doc.org/en/master/usage/extensions/todo.html>`_ that allows for 
inline *todos* in documentation. To enable to inclusion of TODOs in documentation build, set ``todo_include_todos`` to
``True`` in ``conf.py``. When it is set true, all of the todos in the documentation will be listed here.
